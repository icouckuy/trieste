trieste.utils.objectives¶

This module contains toy objective functions, useful for experimentation.

Module Contents¶

branin(x: trieste.type.TensorType) → trieste.type.TensorType[source]¶

The Branin-Hoo function, rescaled to have zero mean and unit variance over \([0, 1]^2\), see here for details.

Parameters

x – Array of two-dimensional points at which to evaluate the function. Shape (…, N, 2).

Returns

The values of the rescaled Branin-Hoo at points in x. Shape (…, N, 1).

Raises

ValueError (or InvalidArgumentError) – If the points in x are not two-dimensional.

BRANIN_GLOBAL_ARGMIN[source]¶

The three global minimizers of the branin() function.

BRANIN_GLOBAL_MINIMUM[source]¶

The global miminum of the branin() function, with shape [1].

mk_observer(objective: Callable[[TensorType], TensorType], key: str) → trieste.observer.Observer[source]¶
Parameters

  • objective – An objective function designed to be used with a single data set and model.

  • key – The key to use to access the data from the observer result.

Returns

An observer returning the data from objective with the specified key.
