trieste.acquisition.combination¶

Module Contents¶

class Reducer(*builders: trieste.acquisition.function.AcquisitionFunctionBuilder)[source]¶

Bases: trieste.acquisition.function.AcquisitionFunctionBuilder

A Reducer builds an AcquisitionFunction() whose output is calculated from the outputs of a number of other AcquisitionFunction()s. How these outputs are composed is defined by the method _reduce().

Parameters

*builders – Acquisition function builders. At least one must be provided.

Raises

ValueError – If no builders are specified.

prepare_acquisition_function(self, datasets: Mapping[str, Dataset], models: Mapping[str, ProbabilisticModel]) → trieste.acquisition.function.AcquisitionFunction[source]¶

Return an acquisition function. This acquisition function is defined by first building acquisition functions from each of the AcquisitionFunctionBuilders specified at __init__(), then reducing, with _reduce(), the output of each of those acquisition functions.

Parameters

  • datasets – The data from the observer.

  • models – The models over each dataset in datasets.

Returns

The reduced acquisition function.
